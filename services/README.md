# AutoContentAI

AI-powered YouTube-to-Shorts automation pipeline.

## Features

- YouTube video downloading
- AI transcription using Whisper
- Viral clip detection using Qwen
- Automatic reel generation
- Subtitle generation
- Vertical short formatting
- Multi-clip support
- Custom reel duration

## Tech Stack

- Python
- FastAPI
- Whisper
- Ollama
- Qwen2.5
- FFmpeg
- yt-dlp
